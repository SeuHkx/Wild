# Nav

- author: Seu

---

导航条的几种方式

## 如何使用

在对应的页面或者模版引入wild.nav.css。部分样式可能会有依赖。

````
 <link rel="stylesheet" href="../../widget/wild.nav.css"/>
````

## 基本结构

````
    <ul class="wd-nav">
        <li>
            <a>...</a>
        </li>
        <li>
            <a>...</a>
        </li>
        ...
    </ul>
````
