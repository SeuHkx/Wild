## hupload 超轻量的强大的上传插件

+ 支持IE7+,以及现代浏览器(支持无耻的IE7:joy:)
+ 支持多个文件,单文件上传
+ 支持所有文件类型上传(也可以通过配置限制文件类型,大小等)
+ 自带进度条(也可以自定义进度条)
+ 支持移动端上传(IOS需要越狱才支持)
+ 支持断点续传(后续更新,测试中)
+ 足以满足日常所需,灵活的参数配置
+ 支持AMD,CommonJS
